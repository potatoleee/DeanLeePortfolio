專案介紹

參與 2022 THE F2E 精神時光屋 Week2 - 今晚，我想來點點簽，
製作一個 Web 版本的簽名服務，挑選喜歡的設計師設計稿進行參賽。
目前支援一頁 PDF 及一個簽名，歡迎使用看看！

網站頁面

  • 上傳頁面：上傳 PDF
  • 創立簽名：透過滑鼠創立個人簽名檔案
  • 編輯頁面：簽名放置於 PDF 上 、下載檔案

使用技術

  • 使用 canvas 製作簽名
  • 使用 PDF.js 在頁面顯示 PDF 資料
  • 使用 fabric.js 讓簽名檔在 PDF 中的拖拉及調整
  • 使用 jsPFD 產出 PDF 檔
  • 引入 設計師製作的 Lottie json 動畫

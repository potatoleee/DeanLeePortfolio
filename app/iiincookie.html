專案發想

從生活周遭取材，
女友因興趣替朋友製作鐵盒餅乾，拍了很多好看的照片，
且有創業的念頭創立了「餅乾生產餡」，
我便希望透過網站吸引第一次認識餅乾生產餡的客人。

網站頁面

  • 電商展示：首頁、商品列表、商品介紹
  • 購物車：刪除、修改商品數量、優惠券、結帳
  • 收藏功能：收藏列表、刪除
  • 關於我們：品牌介紹
  • 最新消息：瀏覽最新消息文章

使用技術

  • Vue 3 Options API + Vite 環境建置
  • 引入 Vue Router 建構路由
  • 引入 Vue Axios 串接 RESTful API 資料
  • Vue Component 元件拆分使用
  • 透過 Pinia 傳遞跨元件資料
  • 使用 localStorage 製作商品收藏功能
  • 使用 GreenSock ScrollTrigger SplitType製作滑鼠滾動特效
  • 引入 Vee Validation 結帳表單驗證
  • Layout 使用 Bootstrap 5 結合 SCSS 切版
  • 使用 Swiper 製作輪播動畫
  • CKEditor 5 後台文章編輯
